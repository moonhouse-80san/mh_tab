다운받은 파일 압축을 풀어서 mh_tab 폴더를 ./modules/editor/components/mh_tab 에 업로더하고

관리자 설정 에디터에서 항목 체크해 주면 됩니다.

 

그럼 게시판 글쓰기 항목에 "탭 박스" 추가됩니다.

2026-07-16 12 50 37.jpg

 

• 탭 박스 — 본문내용에 책갈피 형태의 탭 박스가 생성됩니다.

• 팝업창에서 입력 — 팝업창에서 탭의 제목과 내용을 입력 할 수 있습니다.

• 에디터 창에서 입력 — 에디터 입력 창에서 이미지나 기타 항목을 추가할 수 있습니다.

image.png

• 탭 순서 변경 — 탭의 순서를 임의로 바꿀 수 있습니다.

• 여러 항목 선택 — 탭의 색상, 폰터 크기등을 지정 할 수 있습니다.
